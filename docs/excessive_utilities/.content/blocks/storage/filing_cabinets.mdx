Both **Filing Cabinets** work in the same way: they can store a large amount of a single unstackable item type. Additionally, they do not have a menu, and can only be interacted with via automation.

By default, the regular Filing Cabinet can store up to 270 items, and the Advanced Filing Cabinet can store 540.

## Configuration

Both capacities can be configured.
