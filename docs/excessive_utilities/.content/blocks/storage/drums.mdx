**Drums** store a large amount of a single fluid, and keep their contents when broken.

Their default capacities are:

| Tier | Capacity (mB) |
| ---- | ------------- |
| Stone | 16,000 |
| Iron | 256_000 |
| Reinforced Large | 4,096,000 |
| Demonically Gargantuan | 65,536,000 |
| Bedrockium | 1,048,576,000 |
| Creative | MAX_INT |

## Configuration

The capacities of each tier can be configured.
