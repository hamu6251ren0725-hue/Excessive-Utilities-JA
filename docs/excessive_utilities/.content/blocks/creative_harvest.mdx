The **Creative Harvest** is an uncraftable block that can be used to infinitely mine a block.

Right-click it with a block item to set the block it will mimic. When mined, it will act as if you were mining the mimicked block instead, including dropping what it would drop.

Sneaking will allow you to break the Harvest itself, if you need to move it.
