**Transfer Pipes** are used to connect [Transfer Nodes](@excessive_utilities:transfer_nodes) to destinations, allowing them to transfer items. The same is true for fluids and energy, as well as [Retrieval Nodes](@excessive_utilities:retrieval_node) of all types.

Using a Pipe Wrench on a connection will allow you to block it, disallowing Transfer Nodes from transferring through it.
