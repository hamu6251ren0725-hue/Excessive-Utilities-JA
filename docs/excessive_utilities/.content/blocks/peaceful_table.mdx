The **Peaceful Table** will occasionally simulate killing a monster, inserting their drops into an adjacent inventory. It only works in Peaceful mode, and the inventory must contain a Sword.

Enchantments on the Sword will affect the drops, and the Sword will be damaged as if it were used to kill the mob.

## Configuration

How often the Peaceful Table simulates a kill, as well sa if it works in Peaceful mode, can be configured.
