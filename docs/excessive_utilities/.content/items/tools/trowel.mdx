The **Trowel** is a Shovel that acts as if it has the Silk Touch enchantment when used on shovel-appropriate blocks.

## Configuration

The blocks that the Trowel can be used on are defined in the tag `#excessive_utilities:mineable/trowel`.

By default, this includes all of the following blocks:
- `#minecraft:mineable/shovel`
