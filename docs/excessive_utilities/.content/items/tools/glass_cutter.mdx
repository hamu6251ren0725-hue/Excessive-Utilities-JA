The **Glass Cutter** is used to break Glass blocks as if you had Silk Touch.

It is also used to craft [Ender Shards](@excessive_utilities:ender_shard).

## Configuration

The blocks that the Glass Cutter can be used on are defined in the tag `#excessive_utilities:mineable/glass_cutter`.

By default, this includes all of the following blocks:
- `#minecraft:glass`
- `#minecraft:glass_panes`
