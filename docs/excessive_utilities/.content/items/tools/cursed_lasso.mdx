The **Cursed Lasso** is the alternative to the [Golden Lasso](@excessive_utilities:golden_lasso), but can only hold hostile mobs.

You will only be able to pick up the mob if they at at half health or lower.

## Configuration

Entity types that have the entity type tag `#excessive_utilities:lasso_blacklist` cannot be picked up. By default, this only includes Players.
