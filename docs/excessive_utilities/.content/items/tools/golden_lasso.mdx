The **Golden Lasso** can be used to pick up any non-hostile mobs, storing them in the item. Using it on a block afterwards will place the mob back down.

To pick up hostile mobs, use a [Cursed Lasso](@excessive_utilities:cursed_lasso) instead.

## Configuration

Entity types that have the entity type tag `#excessive_utilities:lasso_blacklist` cannot be picked up. By default, this only includes Players.
