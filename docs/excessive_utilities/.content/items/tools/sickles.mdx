**Sickles** are tools that break blocks like Leaves and Grass in a radius, depending on the tier.

Each tier has the following radius:
- Wooden: 1
- Stone: 2
- Iron: 3
- Gold: 1
- Diamond: 4
- Netherite: 5

## Configuration

The blocks that can be broken by Sickles are defined by the block tag `#c:mineable/sickle`, which contains the following by default:
- `#minecraft:flowers`
- `#minecraft:leaves`
- `minecraft:short_grass`
- `minecraft:tall_grass`
- `minecraft:fern`
- `minecraft:large_fern`
