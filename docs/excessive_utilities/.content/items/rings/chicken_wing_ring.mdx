The **Chicken Wing Ring** allows the holder to slow their fall by holding the jump key while in the air.

While in use, it will occupy 1 [Grid Power]($players/grid_power). Additionally, it can only work while it has charge, which depletes with use and recharges while unused.

## Configuration

The following things can be configured
- GP cost
- Fall speed
- Charge amount and recharge time
