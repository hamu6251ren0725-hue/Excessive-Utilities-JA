The **Ring of the Flying Squid** allows the holder to slowly fly by holding the jump key.

While in use, it will occupy 16 [Grid Power]($players/grid_power). Additionally, it can only work while it has charge, which depletes with use and recharges while unused.

The recipe requires a [Golden Lasso](@excessive_utilities:golden_lasso) holding a Bat and a [Cursed Lasso](@excessive_utilities:cursed_lasso) holding a Ghast.

## Configuration

The following things can be configured
- GP cost
- Flight acceleration and max speed
- Charge amount and recharge time
